# Simple Todo Application (React Frontend)

This repository contains a React frontend application. The current implementation is a lightweight Create React App (CRA) based template with a light/dark theme toggle and responsive styling.

The work item for this repo describes a todo application, but the present code in `react_frontend/src/App.js` is still the starter UI. This README documents the repository as it exists today and explains how to run the frontend.

## Contents

- `react_frontend/`: React application (Create React App)

## Quick start

```bash
cd react_frontend
npm install
npm start
```

Then open:

- http://localhost:3000
